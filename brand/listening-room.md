# THE LISTENING ROOM

Maintained by HYDRO THEORY.

This is not a playlist. It is a reference system — a set of transmissions that define what Phantom Grid sounds like, where it comes from, and what it refuses to compromise. Anyone who understands why every track on this list belongs here understands the label.

## PROTOCOL

HYDRO THEORY maintains this list. It grows slowly. A track is added only when it changes the answer to: *what does Phantom Grid sound like?*

Tracks are never removed. They can be moved between sections.

If a submission references more than three tracks on this list without prompting — it is worth listening to twice.
