# Signal

Phantom Grid is a music label.

It releases electronic music. Machine-precise, intentional, undecorated. Music that has a reason to exist beyond the market. Music that carries something — a position, a weight, an internal logic that holds when the surface is removed.

The genre is not the filter. The position is the filter.

---

The label has no office. No staff in the traditional sense. No A&R who takes meetings. No marketing department. No Notion board where decisions disappear.

Instead: a public Git repository that functions as the label's operating system. Every decision, every rule, every release, every design principle — documented, versioned, traceable. The label runs as code. Its memory is a commit history.

Three AI entities operate within that system. WAVEJUMPER handles visual identity and art direction. HYDRO THEORY handles A&R and curation. STORM SURGE handles communication strategy. Each has a documented aesthetic position, a set of standards, a list of things it will not do.

VOID is the label itself — the position from which the others operate. The silence between the signals.

---

This is not a technology project that makes music as a side effect.

It is a label that uses every available tool — including tools that did not exist five years ago — to do one thing well: find music that has something to transmit, and release it with the care it deserves.

The code is infrastructure. The music is the reason.

---

If you want to understand how it is built, Tobias has documented the full architecture at [tobeworks.de](https://tobeworks.de). Everything is open. Nothing is hidden.

If you want to understand what it sounds like — listen.
